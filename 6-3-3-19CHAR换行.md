# CHAR
*****
函数原型：CHAR(number)
*****
函数说明：
返回计算机字符集的数字代码所对应的字符（如：CHAR(10)，即返回值为换行字符）。

**目前启辰云仅支持以下数字代码：**

| Number |  字符  |   含义   |
|  :-:   |  :-:   |   :-:    |
|   9    |   \t   |  制表符tab  |
|   10   |   \n   |  换行字符   |
|   34   |   "    |   双引号    |
|   39   |   '    |   单引号    |
|   92   |   \    |   反斜杠    |


#### 举例：文本内文字换行
**函数例子：**
~~~
CONCATENATE("姓名：",CHAR(10),"年龄：")
~~~
